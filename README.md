# 项目简介
这个项目是对 M2 上学期 Pytorch 课程的部分总结，除了Pytorch的知识以外，还考虑了同一学期内学到的Monte Carlo课程内容以及完备市场的内容。  
这个项目基于 PyTorch 实现了多资产金融衍生品（篮子看涨期权）的蒙特卡洛定价方法，通过 GPU 加速与在线统计技术显著降低计算复杂度与运行时间，完整实现了随机过程模拟、期权收益计算、置信区间估计以及性能优化。

# 大致研究流程
## 多资产随机过程建模
- 使用多维高斯随机变量模拟生成N条相关资产的价格路径
- 基于Black Scholes模型生成到期资产价格
  
## 计算算术平均篮子看涨期权收益payoff

## 蒙特卡洛模拟与优化
- 使用Numpy与Pytorch分别进行蒙特卡洛模拟，比较运行速率
- 在求样本均值的过程中使用累加的方法降低时间复杂度
- 利用标准正态分布的对称性降低方差从而提升置信区间的精度
